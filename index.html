II. DASAR TEORI

CSS ( Cascading Style Sheets), menggambarkan bagaimana elemen-elemen HTML ditampilkan pada layar, kertas atau media lainnya. CSS mempersingkat pekerjaan dengan cara mengendalikan tampilan (layout) beberapa halaman web sekaligus. CSS dapat ditambahkan kedalam elemen HTML dengan 3 (tiga) cara yaitu : 1.Inline menggunakan style attribute dalam elemen-elemen HTML 2. Internal menggunakan elemen style pada bagian HEAD 3. External menggunakan berkas external CSS

Inline CSS

1.Digunakan untuk menerapakan gaya yang unik pada elemen HTML tunggal.

2.Menggunakan style attribute dari elemen HTML.

3.Contoh dibawah ini, mengatur warna teks dari elemen h1 element menjadi warna biru:

<h1 style="color:blue;">This is a Blue Heading</h1>

This is a Blue Heading

External CSS

Digunakan untuk mendefinisikan gaya (style) dari beberapa halaman HTML.

With an external style sheet, you can change the look of an entire web site, by changing one file!

Untuk menggunakan external style sheet, tambahkan link-nya pada bagian dari halaman HTML, contoh :

<!DOCTYPE html> <html> <head> <link rel="stylesheet" href="styles.css"> </head> <body> <h1>This is a heading</h1> <p>This is a paragraph.</p> </body> </html>

external style sheet dapat ditulis pada semua text editor. Berkasnya tidak harus berisi kode HTML, dan tidak harus disimpan dengan ekstensi “ .css “ .

Berikut adalah contoh tampilan "styles.css" :

body { background-color: powderblue; } h1 { color: blue; } p { color: red; }

Internal CSS

Digunakan untuk mendefinisikan gaya untuk halaman HTML tunggal. Internal CSS didefinisikan pada bagian head dari halaman HTML, dengan elemen style. Contoh :

<!DOCTYPE html> <html> <head> <style> body {background-color: powderblue;} h1 {color: blue;} p {color: red;} </style> </head> <body> <h1>Ini adalah heading</h1> <p>Ini adalah paragraph.</p> </body> </html>

CSS Fonts

Color property mendefinisikan warna teks yang digunakan. Font-family property mendefinisikan huruf (font) yang digunakan. Font-size property mendefinisikan ukuran teks yang digunakan.

Contoh:

<!DOCTYPE html> <html> <head> <style> h1 { color: blue; font-family: verdana; font-size: 300%; } p { color: red; font-family: courier; font-size: 160%; } </style> </head> <body> <h1>This is a heading</h1> <p>This is a paragraph.</p> </body> </html>

The CSS margin property defines a margin (space) outside the border: Potongan CSS margin sebagai berikut :

p { border: 1px solid powderblue; margin: 50px; }

Suatu situs biasanya menampilkan isi/konten kedalam beberapa kolom ( seperti majalah atau surat kabar) . HTML5 menawarkan elemen semantik baru yang menentukan bagian-bagian yang berbeda dari sebuah halaman web

HTML5 Form Elements Elemen <datalist> dari HTML5

Elemen <datalist> menspesifikasikan daftar dari pilihan yang telah ditentukan (pre-defined ) untuk sebuah elemen <input>. Pengguna akan melihat daftar drop-down dari pilihan yang telah ditentukan sebagai input datanya. Atribut list dari elemen <input>, harus mengacu pada atribut id attribute dari elemen <datalist>.

HTML5 <keygen> Element

Tujuan dari elemen <keygen> adalah untuk menyediakan cara yang aman untuk autentikasi para pengguna. Elemen <;keygen> menspesifikasikan daerah pembangkit pasangan kunci (key-pair generator field ) dalam form. Ketika form di kirim (submitted), dua kunci (keys) dibangkitkan, satu sebagai private dan satunya sebagai public. private key disimpan secara lokal, dan public key dikirim ke server. The public key could be used to generate a client certificate to authenticate the user in the future.
